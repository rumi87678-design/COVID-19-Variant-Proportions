# COVID-19-Variant-Proportions
The project is a Server-Side Rendered (SSR) COVID-19 Variant Tracker designed for Bangladesh, built using the Python Flask web framework


A responsive dashboard visualizing COVID-19 variant distribution and genomic surveillance data in Bangladesh. Built with pure HTML and CSS.

Features:
 Real-time variant statistics and percentages
 Interactive timeline showing variant evolution
 Mobile-responsive design
 Bengali language interface with English elements
 Public health alerts and warnings
 Data Source: IEDCR Bangladesh (March 2024)

 
Primary Variant: JN.1 (45% prevalence)
Tech Stack: HTML5, CSS3, Bioinformatic analysis integration

Simply open variant_tracker.html in any browser to view the dashboard. No dependencies required.
