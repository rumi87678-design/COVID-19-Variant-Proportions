# COVID-19-Variant-Proportions
The project is a Server-Side Rendered (SSR) COVID-19 Variant Tracker designed for Bangladesh, built using the Python Flask web framework


A responsive dashboard visualizing COVID-19 variant distribution and genomic surveillance data in Bangladesh. 

This is a static HTML/CSS prototype for a COVID-19 variant tracker dashboard designed for Bangladesh. It's a front-end design mockup that visualizes example data with a focus on a clean layout and Bengali language interface.
